 # 🏥 Medical Landing Page

A modern and responsive landing page for healthcare services.
It highlights medical services, treatment plans, doctors, and booking features with a clean and professional design.

---

## 🚀 Features

* Responsive design for all devices
* Clean and modern UI
* Services and specialties section
* Pricing plans section
* Doctors showcase
* Call-to-action sections (Book a call / Get started)

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
